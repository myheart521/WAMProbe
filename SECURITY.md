# Security policy

## Supported versions

WAMProbe is pre-release software. Security fixes are applied to the latest alpha branch.

## Reporting a vulnerability

Do not publish credentials, private datasets, model tokens, or exploit details in a public
issue. Use GitHub's private vulnerability reporting feature for this repository. If it is
not available, open a minimal issue requesting a private maintainer contact without including
the sensitive details.

WAMProbe is a research evaluation tool, not a certified robot safety system. Its metrics and
action-selection demos must not be treated as a safety controller for real hardware.
